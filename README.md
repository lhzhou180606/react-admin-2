## ✨ 特性

- 丰富业务组件，除了基础的UI组件外，还提供常用的业务组件示例
- 海量页面模板，提供多种场景页面，如：页面、布局、应用等
- 高度可自定义，减少组件间耦合度，允许开发者自定义或重新封装组件
- 积极的技术栈，积极跟进主流技术栈，依赖快速更新，高效完成业务功能

## 🍭 使用

```bash
$ git clone https://github.com/shulkme/react-admin.git
$ cd react-admin
$ pnpm install
$ pnpm dev
```
